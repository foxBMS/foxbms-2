.. _fox_cli_package:

|fox-cli-package|
-----------------

.. attention::

   Ensure that the |virtual-python-environment|, which contains the
   |fox-cli-package|, is :ref:`activated<INSTALL_VENV>` before using it.

.. note::

   The ``fox_cli`` directory within the user's Documents directory will contain
   any data generated by the |fox-cli-package|.

The |fox-cli-package| is a condensed version of the |fox-cli|.
Instead of using the shell wrappers, which only work within the repository,
the ``fox-cli`` command is provided to interact with the package.
This command is only accessible outside of the repository.

.. tabs::

   .. group-tab:: Win32

      .. code-block:: shell

         fox-cli

   .. group-tab:: Linux

      .. code-block:: shell

         fox-cli
