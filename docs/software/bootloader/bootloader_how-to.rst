How to update the foxBMS application using the bootloader
=========================================================

#. Connect the foxBMS 2 hardware via a CAN-USB converter to the host computer
#. Make sure no other component except the BMS is active on the CAN bus
#. Start the bootloader update process on the host computer
#. Power-cycle the BMS
#. The update program should now be executed.
#. Power-cycle the BMS
